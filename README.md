# Online Shopping - Backend + Frontend scaffold

1. Open terminal in c:\Users\bhata\OneDrive\Desktop\online shopping\ecommerce-web-app
2. Install:
   npm install
3. Create .env (already added). Ensure MySQL is running and database online_shopping exists.
4. Start:
   npm run dev
5. Open http://localhost:4000

Notes:
- Tables required (do NOT rename): admin_activity, admin_permissions, cart, categories, order_items, orders, payments, products, user_history, users, wishlist
- Replace JWT_SECRET in .env before production.
